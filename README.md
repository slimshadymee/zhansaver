# InstaLoader — Node.js сервер

Локальный сервер для скачивания фото, видео и каруселей из Instagram.

---

## Требования

- [Node.js](https://nodejs.org/) версии 16 или выше

---

## Установка и запуск

```bash
# 1. Перейди в папку проекта
cd instaloader-server

# 2. Установи зависимости
npm install

# 3. Запусти сервер
npm start
```

После запуска открой браузер и перейди на:
```
http://localhost:3000
```

## Как это работает

Сервер пробует 3 метода по очереди:

1. **Instagram ?__a=1 API** — самый быстрый
2. **HTML парсинг** — парсит страницу как браузер
3. **GraphQL API** — запасной вариант

Все запросы к Instagram идут с сервера с заголовками реального браузера.
Медиафайлы проксируются через `/proxy?url=...` — браузер скачивает их напрямую с нашего сервера.
